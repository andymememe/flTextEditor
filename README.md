# flTextEditor
**flTextEditor** is a text editor using **FLTK** as widget toolkit.

## Requirement
### Windows
1. **FLTK 1.3.x** library compiled by **MinGW**
2. **MinGW**

## Binary Release
[Windows](https://github.com/andymememe/flTextEditor/releases)

## Installation & Execution
### From Source
#### Windows
```shell
make
```

### From Binary
#### Windows
```shell
flTextEditor.exe
```

## Usage
Only support **txt** file
